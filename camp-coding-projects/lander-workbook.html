Lander

Build this game in PixelPad one step at a time. After each step, press ▶ Play to run your code and see what changed.

Tip: tap Print to PDF (top-right) for a printable copy.

Before you start:
1. Open a new PixelPad project.
2. Add these pictures (Sprites). Draw each on a pixel canvas of the size shown (width×height): Tap a sprite to open the drawing app at the right size.
   • rocket.png — draw it 30×46, orange ↗
   • pad.png — draw it 170×20, green ↗
   • bg.png — draw it 760×560, dark ↗
3. Add these Classes: Bg, Pad, Rocket.
4. Add these Rooms: Play, GameOver.
Now follow the steps. Type only the green lines each time.
